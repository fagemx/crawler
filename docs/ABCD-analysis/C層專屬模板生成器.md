# C層專屬模板生成器

## 功能
根據帳號貼文樣本分析結果，生成該帳號專屬的C層（節奏與韻律規律）規則模板

## 使用說明

1. **樣本輸入**：提供該帳號5-10篇代表性貼文
2. **節奏分析**：系統提取該帳號的C層特徵
3. **模板生成**：產出該帳號專屬的C層規則
4. **節奏應用**：使用專屬模板創作新內容
5. **效果驗證**：檢核生成內容的節奏和語感
