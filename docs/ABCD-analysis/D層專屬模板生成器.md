# D層專屬模板生成器

## 功能
根據帳號貼文樣本分析結果，生成該帳號專屬的D層（模板骨架與肌肉生成法）規則模板

## 核心理念
**D層的目的是將內容注入「私語」與「感官真實感」，模擬該帳號真實的內心驅動和自然表達方式，讓生成內容具備人性化的溫度和即時感。**
